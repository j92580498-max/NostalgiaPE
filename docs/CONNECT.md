# Подключение к серверу `nostalgiape.ddns.net:19132`

## Шаги

1. Запустите **NostalgiaPE**.
2. На стартовом экране заполните поля (значения по умолчанию уже подставлены):
   - **Имя игрока (Name):** любой ник, например `Player`.
   - **Адрес сервера (Server):** `nostalgiape.ddns.net`
   - **Порт (Port):** `19132`
3. Нажмите **Подключиться (Join)**.
4. Вверху экрана появится статус подключения:
   - `Connecting to server...` — идёт RakNet-хендшейк;
   - `Handshake OK, logging in...` — соединение установлено, отправлен логин;
   - `World starting...` — сервер прислал `StartGame`;
   - `In game` — вы в мире, чанки подгружаются.

## Что работает после входа

| Действие | Как |
|---|---|
| **Осмотреться** | Проведите пальцем по экрану (перетаскивание камеры). |
| **Ходить** | Крестовина слева внизу (вперёд/назад/влево/вправо). |
| **Прыгать** | Кнопка **JUMP** справа. |
| **Сломать блок** | Наведите центр экрана на блок → **BREAK**. |
| **Поставить блок** | Наведитесь на грань → **PLACE** (ставит камень). |
| **Чат** | Кнопка **CHAT**, введите сообщение, «Send». Входящие сообщения видны слева. |

Позиция игрока автоматически отправляется на сервер (`MovePlayer`) примерно 20
раз в секунду, поэтому другие игроки на сервере видят ваше перемещение.

## Как это работает под капотом

Клиент реализует классический сетевой стек Pocket Edition:

```
UDP  →  RakNet (structure 5)  →  MCPE 0.8.x game packets
```

1. **Offline-хендшейк RakNet:**
   `OPEN_CONNECTION_REQUEST_1/2` ↔ `..._REPLY_1/2` (с 16-байтовым MAGIC
   `00 ff ff 00 fe fe fe fe fd fd fd fd 12 34 56 78`).
2. **Online-сессия:** `CLIENT_CONNECT → SERVER_HANDSHAKE → CLIENT_HANDSHAKE`,
   после чего игровые пакеты идут внутри надёжно-упорядоченных
   `DATA_PACKET`-датаграмм.
3. **Игровой логин:** `LoginPacket` (ник + protocol 14) → сервер отвечает
   `LoginStatus` и `StartGame`.
4. **Мир:** сервер шлёт `FullChunkData` (сжатые zlib блоки), клиент их
   распаковывает и строит воксельный меш.

Полное описание формата пакетов — в [`PROTOCOL.md`](PROTOCOL.md).

## Если не подключается

- Убедитесь, что телефон в интернете и порт **19132/UDP** не блокируется сетью.
- Проверьте, что сервер `nostalgiape.ddns.net` сейчас онлайн.
- Если статус завис на `Connecting...` дольше ~15 секунд — сервер недоступен
  или использует другую версию протокола. Клиент повторяет хендшейк
  автоматически раз в секунду.
