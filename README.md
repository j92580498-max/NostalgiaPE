# NostalgiaPE — клиент для классических серверов Minecraft PE Alpha 0.8.x

**NostalgiaPE** — это полностью самостоятельный, чистый (clean-room) клиент для
Android, написанный «с нуля» и вдохновлённый **Minecraft Pocket Edition
Alpha 0.8.1**. Он не содержит и не использует оригинальный код, текстуры,
модели, звуки или иные материалы Mojang/Microsoft — весь код, палитра блоков и
графика написаны заново на основе **открыто задокументированного сетевого
протокола** классических серверов PE (семейство PocketMine / NostalgiaCore).

Главная цель проекта — современный клиент, совместимый со старыми серверами
Minecraft PE Alpha 0.8.x, в первую очередь с **`nostalgiape.ddns.net:19132`**.

> ⚠️ **О товарных знаках и материалах.** «Minecraft» — товарный знак Mojang AB.
> Этот проект не аффилирован с Mojang/Microsoft и не распространяет их ресурсы.
> Совместимость достигается за счёт реализации сетевого протокола, а не за счёт
> копирования игрового кода или ассетов.

## Что уже работает

| Возможность | Статус |
|---|---|
| RakNet-хендшейк (offline + online) со «structure 5» серверами | ✅ |
| MCPE-логин (протокол 0.8.x, `LoginPacket` / `LoginStatusPacket`) | ✅ |
| Приём мира: `FullChunkData` (zlib), распаковка блоков XZY 16×128×16 | ✅ |
| `StartGame` / спавн игрока, `Ready`-пакет | ✅ |
| Рендер вокселей на OpenGL ES 2.0 (отсечение граней, «свет» по граням) | ✅ |
| Движение игрока: гравитация + AABB-коллизии, прыжок | ✅ |
| Отправка `MovePlayer` на сервер (~20 раз/с) | ✅ |
| Ломание блока (`RemoveBlock`) и установка (`UseItem`) | ✅ |
| Чат: приём `Message`, отправка `Chat` | ✅ |
| Сенсорное управление: джойстик, обзор перетаскиванием, кнопки | ✅ |
| Обновление одиночных блоков `UpdateBlock` | ✅ |

Клиент задуман как «наблюдающий/играющий» на классических серверах: он корректно
входит, получает и отображает мир, двигается, строит/ломает и общается в чате.
Инвентарь, крафт, мобы и звуки в текущей версии не реализованы (см. «Дорожная
карта»).

## Архитектура

```
app/src/main/java/com/nostalgiape/client/
├── net/
│   ├── RakNetClient.java     — транспорт RakNet (хендшейк, инкапсуляция, ACK)
│   ├── McpeProtocol.java     — идентификаторы игровых пакетов 0.8.x
│   └── GameSession.java      — игровой слой: логин, чанки, движение, чат
├── world/
│   ├── Block.java            — id блоков + оригинальная палитра цветов
│   └── World.java            — разреженный воксельный мир (чанки 16×128×16)
├── render/
│   ├── WorldRenderer.java    — рендер OpenGL ES 2.0
│   └── ChunkMesh.java        — построение меша чанка с отсечением граней
├── entity/
│   └── LocalPlayer.java      — позиция, физика, raycast для строительства
├── util/
│   ├── ByteBuf.java          — чтение/запись бинарных данных (BE + triads)
│   └── Mat4.java             — матрицы для GL-конвейера
└── ui/
    ├── TitleActivity.java    — экран ввода сервера/ника
    └── GameActivity.java     — игровой экран + HUD + чат
```

Подробнее о протоколе — в [`docs/PROTOCOL.md`](docs/PROTOCOL.md).

## Быстрый старт

1. Установите готовый APK: [`dist/NostalgiaPE-1.0-debug.apk`](dist/NostalgiaPE-1.0-debug.apk).
2. Откройте приложение. Поля уже заполнены значениями по умолчанию:
   - **Ник:** `Player`
   - **Сервер:** `nostalgiape.ddns.net`
   - **Порт:** `19132`
3. Нажмите **Подключиться (Join)**.

Полные инструкции: [`docs/BUILD.md`](docs/BUILD.md) и
[`docs/CONNECT.md`](docs/CONNECT.md).

## Сборка

Требуется Android Studio (Giraffe+ / AGP 8.5) или командная строка с Android SDK.

```bash
git clone <repo-url> NostalgiaPE
cd NostalgiaPE
./gradlew assembleDebug
# → app/build/outputs/apk/debug/app-debug.apk
```

Проект открывается и собирается в Android Studio без дополнительной настройки
(`File → Open → выбрать папку NostalgiaPE`). Подробности — в
[`docs/BUILD.md`](docs/BUILD.md).
